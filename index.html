¿Porque aprender git?

Git es una herramienta que nos permite controlar el flujo de trabajo de nuestro código mediante sus distintos comandos nos permite:

Crear espacios seguros mediante ramas para hacer cambios sin afectar el código en producción

Clonar el código principal y tener una copia local

Fusionar nuestro código al principal

Nos permite llevar un control de quien hizo un cambio y cuando lo hizo

Asi que por esto deberias aprender Git
